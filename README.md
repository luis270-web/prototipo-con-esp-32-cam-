# prototipo-con-esp-32-cam-
desarrollo de un prototipo con una esp 32 cam entrenado con dos objetos diferentes que al momento de reconocer uno mandar su nombre a travez de una pantalla oled, prendiendo un led y que si detecta un segundo objeto prendera un led diferente a menos que no detecte nada solo prendera un led azul, todo esto documentado en este repositorio.
